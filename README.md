##Description
This sample application shows the workflow to create custom activity and custom apppackage using Java for AutoCAD.IO V2 API.

##Dependencies

IntelliJ IDEA 14.
OData 4.0 Java library.

##Setup/Usage Instructions

Please refer to [AutoCAD.IO V2 API documentation](https://developer.autodesk.com/api/autocadio/v2/#tutorials).
